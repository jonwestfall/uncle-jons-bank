# Roles

## Parent

Parents (and guardians) can:

- Create and manage child accounts.
- Post credits/debits, chores, recurring items, loans, and CDs (based on permissions).
- Review and approve or deny child withdrawal requests.
- Send messages to children.

## Child

Children can:

- Log in with access code credentials.
- View balance, ledger, recurring activity, and CD offers.
- Request withdrawals.
- Request/accept/decline loans where allowed.
- Complete chores, redeem coupons, and use learning modules.

## Admin

Admins can:

- Access `/admin` routes and global management views.
- Manage users, children, and transactions platform-wide.
- Manage site settings (site name, URL, currency, rates, fees).
- Manage role permissions.
- Broadcast announcements.

## Notes

- Parent-child links include per-child permissions.
- Child accounts have a reduced surface and cannot use parent/admin routes.
