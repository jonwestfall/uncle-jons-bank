# Common Tasks

## Parent tasks

- Add a child: Dashboard -> add child.
- Share access: Child -> share code -> send code to guardian.
- Approve withdrawal: Withdrawals panel -> approve/deny.
- Add recurring charge or credit: Recurring panel -> create item.

## Child tasks

- View account summary: Child dashboard.
- Request withdrawal: Withdrawals section -> submit request.
- Redeem coupon: Child coupons page -> enter code.
- Review loan status: Child loans page.

## Admin tasks

- Manage users: `/admin` -> users.
- Edit site settings: `/admin` -> settings modal.
- Review all transactions: `/admin` -> transactions.
- Broadcast message: Messaging -> broadcast.
